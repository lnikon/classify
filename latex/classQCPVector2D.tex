\doxysubsection{Detailed Description}
Represents two doubles as a mathematical 2D vector. 

This class acts as a replacement for Q\+Vector2D with the advantage of double precision instead of single, and some convenience methods tailored for the \mbox{\hyperlink{classQCustomPlot}{Q\+Custom\+Plot}} library.
